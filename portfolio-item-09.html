Pomodoro Clock With React

tic, tac, tic, tac

This is one of the projects I did for the freecodecamp front end certification.

I really like this project, I use it practically daily to be able to take breaks while I work, since I usually lose track of time completely, so this project helps me not to be in front of the computer all the time and to be able to rest my eyes.

although surely a better use would be for physical activity, in any case, you can choose the time you want for each session and rest!

By the way, this project is also made entirely with ReactJs.

If you want you can take a look at the code in code pen!

Or you can use it by clicking in the github pages link!
