React Hooks Theremmin

a very cool instrument!

Music is one of my great passions. I have a wacom tablet to draw in digital format, and one day it occurred to me that I could use the surface of it to play a theremin, remembering a similar project that I saw a couple of years before, so I decided to create my own theremin.

For this project I use ReactJs and ReactDOM, I also use styled-components, styled-reset and tone. I had wanted to try styled components for a long time and it was really nice! also, tone made it very easy for me to work with the sound ranges and create the wave itself!

If you want you can take a look at the code in the github repository!

Or you can use the theremin itself in the github pages link!
